# О шаблоне

**[▶ Запустить в Colab](https://colab.research.google.com/github/vokoder1990/ml_practice/blob/main/preprocessing/stroke_prediction/preprocessing_v4_github_stroke.ipynb)**


Структура шаблона создана мной на основе теории курса Google [Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course). Для генерации кода по созданной структуре использовался DeepSeek.

**Что сделано мной:**
- Структура ноутбука (блоки, порядок выполнения)
- Система универсальных настроек для будущего шаблона
- Логика обработки пропусков, диапазонов и редких категорий
- Анализ данных (что удалять, что оставлять, почему)
- Отчёты и документация
- Перекрестная проверка блоков кода через разные LLM

**Репозиторий:**

Шаблон применяется к разным датасетам. Каждый датасет в отдельной папке с настроенным ноутбуком и отчётом.
